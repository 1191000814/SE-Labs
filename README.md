## 项目特点

**高度原生构建**

前端的同学使用html + css + js构建

后端直接使用javaee构建

(几乎无任何代码的复用, html和css)

**完全的前后端分离**

前端的同学没有看后端的代码

后端的同学也没有看前端的页面

甚至不知道对方使用什么语言实现的…

**完全依赖于接口文档**

接口文档里里提供一切开发所需的规范和信息，由前后端共同商议
